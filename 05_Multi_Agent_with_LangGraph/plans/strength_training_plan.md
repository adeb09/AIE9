# Strength Training Plan Overview

**Frequency:** 4 days per week  
**Duration:** 3 months  
**Split:** Upper/Lower Body

## Weekly Schedule
- **Day 1:** Upper Body
- **Day 2:** Lower Body
- **Day 3:** Rest or Active Recovery
- **Day 4:** Upper Body
- **Day 5:** Lower Body
- **Day 6:** Rest or Active Recovery
- **Day 7:** Rest

## Month 1: Foundation Building
**Upper Body:**
1. Bench Press: 3 sets of 8-10 reps
2. Bent-over Rows: 3 sets of 8-10 reps
3. Shoulder Press: 3 sets of 8-10 reps
4. Pull-ups or Lat Pulldowns: 3 sets of 8-10 reps
5. Bicep Curls: 3 sets of 10-12 reps
6. Tricep Dips: 3 sets of 10-12 reps

**Lower Body:**
1. Squats: 3 sets of 8-10 reps
2. Deadlifts: 3 sets of 8-10 reps
3. Lunges: 3 sets of 10-12 reps (each leg)
4. Leg Press: 3 sets of 8-10 reps
5. Calf Raises: 3 sets of 12-15 reps

## Month 2: Increasing Intensity
**Upper Body:**
1. Increase weights by 5-10% from Month 1.
2. Add 1 additional set to each exercise.

**Lower Body:**
1. Increase weights by 5-10% from Month 1.
2. Add 1 additional set to each exercise.

## Month 3: Peak Phase
**Upper Body:**
1. Focus on heavier weights (increase by another 5-10%).
2. Incorporate drop sets for the last set of each exercise.

**Lower Body:**
1. Focus on heavier weights (increase by another 5-10%).
2. Incorporate drop sets for the last set of each exercise.

## Additional Tips
- **Warm-Up:** Always start with a 5-10 minute warm-up (e.g., light cardio, dynamic stretches).
- **Cool Down:** Finish with stretching to improve flexibility and recovery.
- **Nutrition:** Ensure a protein-rich diet to support muscle growth.
- **Rest:** Allow adequate recovery time between workouts.
